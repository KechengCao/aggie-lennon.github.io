SDL_RWops

SDL_RWops

参考链接：Click to jump

作用

为流 I/O 提供抽象接口的结构

数据字段

Sint64 (*)(SDL_RWops *) size size 是一个函数指针，用于报告流的总大小（以字节为单位）。如果无法确定流大小（或者因为它对于流类型没有意义，或者存在错误），则此函数返回 -1。
Sint64 (*)(SDL_RWops *, Sint64, int) seek seek 是一个函数指针，用于定位流中的下一个读/写操作。这会寻找字节偏移量。如果流无法查找（或者因为它对流类型没有意义，或者出现错误），则此函数返回 -1，否则返回新位置。
size_t (*)(SDL_RWops *, void *, size_t, size_t) read read 是从流中读取的函数指针。它最多将 num 个对象读取到 ptr 指向的缓冲区指针中，每个对象的大小为字节。返回读取的对象数量，该数量可能少于请求的数量。出错或 EOF 时返回 0。
size_t (*)(SDL_RWops *, const void *, size_t, size_t) write write 是写入流的函数指针。它从缓冲区指针到 ptr 精确写入每个 size 字节的 num 个对象。返回写入的对象数量，出错时该数量将少于请求的数量。
int (*)(SDL_RWops *) close close 是一个清理流的函数指针。它应该释放流使用的所有资源，并使用 SDL_FreeRW() 释放 SDL_RWops 本身。如果成功，则返回 0；如果流无法刷新到磁盘（或其他位置），则返回 -1。即使刷新流失败，SDL_RWops 在此调用后也不再有效。
Uint32 type 流类型
union hidden 特定类型的数据

Seek Function

Identifier Value Explanation
RW_SEEK_SET 0 从数据开头查找
RW_SEEK_CUR 1 相对于当前读取点进行查找
RW_SEEK_END 2 相对于数据末尾查找

Stream Type

Identifier Value Explanation
SDL_RWOPS_UNKNOWN 0 未知的流类型或应用程序定义的
SDL_RWOPS_WINFILE 1 Win32 文件句柄
SDL_RWOPS_STDFILE 2 stdio.h 文件*
SDL_RWOPS_JNIFILE 3 Android asset
SDL_RWOPS_MEMORY 4 内存流（读/写）
SDL_RWOPS_MEMORY_RO 5 内存流（只读）
